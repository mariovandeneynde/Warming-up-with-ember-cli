Codeschool's Warming Up With Ember converted to Ember CLI

# Step 1: Install Ember CLI

# Step 2: Create a new app
- create folder
- ember new 'my-app'
- go to the new 'my-app' folder
- copy the app and dist folder from github to your new app
- ember serve

end start coding :-)
